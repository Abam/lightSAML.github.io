LightSAML

LightSAML is a stack of PHP libraries and bundles implementing OASIS' SAML 2.0 protocol.

  • LightSAML Core

    A core library implementing data model, XML serialization/deserialization, bindings, and profiles.

  • LightSAML IDP

    A library implementing IDP side of the Web Browser Single SignOn SAML profile.

  • LightSAML Logout

    A library implementing the Web Browser Single Logout SAML profile.

  • Symfony Bridge

    A Symfony bridge bundle configuring internal profile services.

  • Symfony SP Bundle

    A Symfony bundle implementing Service Provider side of the Web Browser Single Sign On SAML profile as a Symfony security listener.
